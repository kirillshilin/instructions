# Markdown Formatting

Write short, direct sentences.

- Do not wrap text manually. Let the editor wrap.
- Break lines only for structure.
- Use short, clear headings.
- Prefer concise bullets over long paragraphs.
- Prefer instructions over background.
- Use fenced blocks for multi-line code.
- Use inline code for commands, paths, and identifiers.
